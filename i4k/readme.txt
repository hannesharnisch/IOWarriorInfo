Klassenbibliothek um mit den IO-Warrior zu arbeiten.


Verzeichnisstruktur

-doc 
  Enthält die Java-Doc Dokumentation

- install
  Enthält die rules datei um die USB-Rechte zu setzen
  Enthält ein minimales Testprogramm für Eclispe

- lib32 
  Bibliotheken für 32-Bit Systeme

- lib64
  Bibliotheken für 64-Bit Systeme

Eine der Beiden (lib) Verzeichnisse muss eingebunden werden!
